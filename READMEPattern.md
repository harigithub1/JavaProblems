🧠 The 20 Core Patterns
🟢 Arrays & Basic Techniques
1) Two Pointers
Opposite ends / slow-fast
Example: remove duplicates, container with most water
2) Sliding Window
Fixed or variable window
Example: longest substring, max sum subarray of size k
3) Prefix Sum
Cumulative sums for O(1) range queries
Example: subarray sum equals k
4) Kadane’s Algorithm
Max/min subarray
Example: maximum subarray
5) Array Manipulation / Simulation
In-place tricks
Example: rotate array, product except self
🔵 Searching & Sorting
6) Binary Search (Classic)
Sorted arrays
Example: find element
7) Binary Search on Answer
Search space = answer range
Example: capacity to ship packages
8) Sorting + Greedy
Sort first, then decide
Example: meeting rooms, intervals
🟡 Hashing & Data Structures
9) HashMap / HashSet
Frequency counting, lookup
Example: two sum, anagram
10) Heap / Priority Queue
Top K / min-max tracking
Example: kth largest element
11) Monotonic Stack / Queue
Next greater/smaller element
Example: daily temperatures
🟣 Linked Lists
12) Linked List Manipulation
Reverse, merge, cycle detection
Example: reverse linked list
🌳 Trees & Graphs
13) Tree DFS (Recursion)
Pre/in/post order
Example: max depth
14) Tree BFS (Level Order)
Queue-based traversal
Example: level order traversal
15) Graph DFS/BFS
Traversal, connected components
Example: number of islands
16) Topological Sort
DAG ordering
Example: course schedule
17) Union Find (Disjoint Set)
Connectivity problems
Example: redundant connection
🔴 Backtracking & Recursion
18) Backtracking
Try all combinations
Example: subsets, permutations
🟠 Dynamic Programming
19) 1D / 2D Dynamic Programming
Memoization / tabulation
Example: climbing stairs, knapsack
⚫ Specialized / Advanced
20) Bit Manipulation
XOR tricks, masks
Example: single number
